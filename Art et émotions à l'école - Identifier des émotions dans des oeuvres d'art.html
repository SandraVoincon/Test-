Identifier des émotions dans des oeuvres d'art


Notre Ressource Educative Libre (REL) sur Wooclap.com

L'un des domaines d'apprentissage du cycle 1 est "Agir, s'exprimer, comprendre à travers les activités artistiques".
Notre Ressource Educative Libre (REL) amène les élèves à identifier les émotions de base (colère, peur, joie, tristesse, surprise) dans différentes oeuvres d'art.

Voici notre REL


Qu'est-ce-que Wooclap.com ?


Le site Wooclap est créé en 2015 par Sébastien Lebbe et Jonathan Alzetta.
Il s'agit d'un système de questions/réponses qui permet d'interagir avec le public via ordinateur, tablette ou smartphone. L'application permet de collecter et traiter instantanément des réponses transmises.
Wooclap propose une vingtaine d'outils : questions à choix multiples, échelle, sondage, trouvez sur l’image, nuage de mots, questions ouvertes, devinez un nombre, priorisation, ordonnancement, appariement, texte à trous, diapositive, brainstorming, vidéo, etc. 

Comment récupérer notre REL ?

Pour se faire munissez-vous de ce code : ASRBCL. Ensuite, vous aurez besoin de vous connecter sur Wooclap.com



Pour vous connectez sur Wooclap vous devrez fournir une adresse mail, un identifiant et un mot de passe. Pour avoir notre REL dans vos évènements Wooclap, sélectionnez Importer un évènement.








Puis entrez notre code. C'est ainsi que vous pourrez importer un grand nombre de ressources à modifier, vous n'aurez qu'à récupérer leur code de partage !









Vous obtenez finalement une copie de notre REL, que vous pouvez entièrement modifier, avec un nouveau code à partager. Vous pouvez tout modifier : la mise en page, les questions, les oeuvres utilisées, le sujet de la ressource. Vous pouvez tout faire.
